# 信息级别

* 网站
  * 模块、子模块
    * 页面



# 流量信息

* PV：Page View，网站浏览量。每当一个页面被打开，就加一。
* UV：Unique Visitor，独立访客量。指一天内访问某站点的人数。通常以cookie为依据。
* IP：独立IP数。
* VV：Visit View，独立访客的访问次数。当访客完成所有浏览并最终关闭该站点的所有页面时，视为完成一次访问。
* 随时间的变化趋势

* 访客标签
